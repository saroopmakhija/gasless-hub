## Requirements

- [FNM](https://github.com/Schniz/fnm) or [NVM](https://github.com/nvm-sh/nvm)
- [Node](https://nodejs.org)
- [Bun](https://bun.sh)

## Installation

```bash
git clone git@github.com:samui-build/samui-wallet.git
cd samui-wallet
fnm use # or nvm use
bun install
bun dev # or bun --filter=<app-name> dev
```
